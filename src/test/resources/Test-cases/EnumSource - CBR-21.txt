Поиск организаций по региону должен показать не пустой список
ID: CBR-21

Предусловия:
- открыта страница https://www.cbr.ru/finorg/

Шаги:
1. Выбрать тип - "икрофинансовые организации"
2. Выбрать статус вида деятельности "действующий"
3. Выбрать регион ("Субъект РФ") один из списка
3. Нажать кнопку "Искать"

ОР - Список организаций не пустой
